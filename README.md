ask Tracker MERN App

A simple Task Tracker Web Application built using the MERN stack.

Live Links

Frontend: https://task-tracker-mern-five.vercel.app

Backend API: https://task-tracker-mern-9fla.onrender.com/api/tasks

GitHub Repo: https://github.com/arpit-yadav01/task-tracker-mern

Backend
cd backend
npm install
npm run dev

Frontend
cd frontend
npm install
npm run dev


Environment Variables

Backend

MONGO_URI=your_mongodb_url
PORT=5000


Frontend

VITE_API_BASE_URL=https://task-tracker-mern-9fla.onrender.com/api
